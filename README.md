## FAQ

### ChatGPT Plus 支持支付宝或微信吗？

OpenAI 官方付款页通常不直接支持国内支付宝或微信。这里介绍的是通过第三方卡密方式完成充值。

### 为什么要复制 Session？

Session 用于识别当前登录的 ChatGPT 账号。复制前一定要确认登录的是目标账号。

### 会不会充错账号？

如果浏览器登录了多个账号，就有可能出错。建议充值前退出无关账号，确认页再核对邮箱。

### 充值完成后多久能用？

网页端通常较快显示，移动端或客户端可能需要等待同步。可以刷新或重新登录。

### 卡密找不到怎么办？

先回到订单页面查看发货信息，不要马上重复购买。保留付款记录后联系售后查询。
